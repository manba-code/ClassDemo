## 5. 仍存在的覆盖缺口

### 🔵 低优先级（可接受）

| 类/场景 | 未覆盖原因 | 建议 |
|---------|-----------|------|
| `NetworkApplication.main()` | 启动入口 | 集成测试或手动验证 |
| `WebMvcConfig` | CORS 配置类 | `@SpringBootTest` 集成测试 |
| `MybatisPlusConfig` | MyBatis 插件配置 | 集成测试覆盖 |
| `KnowledgePointService` pageSize==0 分支 | normalize 逻辑保护，实际不可达 | 无需补充 |
| `LayerInfoConverter` 部分 JSON 异常分支 | 已有 Converter 测试覆盖主路径 | 可选补充 |
| `KnowledgeUpdateRequest.hasAnyField()` 单分支 | DTO 逻辑分支 | 影响极低 |
